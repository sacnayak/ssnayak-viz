About the data quality

Overview

The data set in general was of good quality. I would've liked more public datasets with respect policy etc in each of these states. Also, I would've liked more detail on the incident - if it was a homicide, suicide, accidental shooting, police shooting etc. Adding that element to the visualization would've spruced it up a bit.

Data cleaning and processing for D3

I had to write a clean_up.py script (available on my github) to filter out the data into a form that was good for visualizing in my calendar. While the data was raw and listed out each incident on different days, I had to summarize it with respect to count of incidents on each day.

Datetime issues
